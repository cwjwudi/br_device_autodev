Manual Paper transform test commands
====================================

Write any one of these complete JSON strings to SVG:strTransform for a single-part test.
Set SVG:astSvgPu[0].stCmd.bManualTestMode = 1 before writing these commands, otherwise ProgramCyclic will overwrite strTransform.
Set SVG:astSvgPu[0].stCmd.bManualTestMode = 0 to return to automatic command-bit control.

1. emergency - red  修改
[{"select":"#emergency","duration":0,"style":"fill:3"}]


2. emergency - normal gray  修改
[{"select":"#emergency","duration":0,"style":"fill:1"}]

3. line_1 - show 通过
[{"select":"#line_1","duration":0,"style":"display:inline"}]


4. line_1 - hide 通过
[{"select":"#line_1","duration":0,"style":"display:none"}]

5. line_2 - show  通过
[{"select":"#line_2","duration":0,"style":"display:inline"}]

6. line_2 - hide 通过
[{"select":"#line_2","duration":0,"style":"display:none"}]

7. fan1 - rotate once around local center (0,0)   通过
[{"select":"#fan1","duration":1000,"spin":[360,0,0]}]

8. fan1 - stop/no rotation command   通过
[{"select":"#fan1","duration":0,"spin":[0,0,0]}]

9. roller_up - move down 20 units 修改
[{"select":"#roller_up","duration":1000,"translate":[168, 470]}]

10. roller_up - return home 修改
