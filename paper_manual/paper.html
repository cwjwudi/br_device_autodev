Paper

Widget "Paper" provides an area that can be used to display and modify SVG elements.

It can be used, for example, to visualize a complete machine process (on an injection molding machine, for example).

zIndex

UInteger

zIndex determines the z-order of the widgets. For overlapping widgets, zIndex determines which widget is above and which is below. The widget with the larger zIndex is above; the one with the smaller zIndex is below. zIndex must be unique in a piece of content or container widget. zIndex is assigned automatically by the visual editor and can be modified in this view via the toolbar.
