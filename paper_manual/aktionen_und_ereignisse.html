Actions and events

The following section describes the possible actions and events of the widget.

Actions

Name

Description

Arguments

ZoomIn

Displays the current drawing area zoomed in by 20%.

  

ZoomOut

Displays the current drawing area zoomed out by 20%.

  

ZoomReset

Rests the zoom for the characters to the standard view.

  

SetVisible

Controls the visibility of the widget.

  • value: Boolean

    Visible if the value is true. Otherwise, invisible.

SetEnable

Controls the usability of the widget.

  • value: Boolean

    Enabled if the value is true. Otherwise, disabled.

SetStyle

Applies a predefined style to the widget.

  • value: StyleReference

ShowTooltip

This action can be used to enable tooltip mode for the widget. When enabled, a tooltip indicator is displayed inside the widget. For this, a text must be configured on property "tooltip". This text is displayed when the indicator is selected. Any other operation of a widget exits tooltip mode.

  

Focus

Sets the focus to the widget when keyboard operation is enabled.

Events

Name

Description

Arguments

TransformDone

Triggered when the current transformation is completed.

  • select: String

    "select" returns parameter "select" of the "transform" value that was the trigger for this event.

Click

Triggered when the widget is clicked.

  • x: Integer

    Returns the x-coordinate of the click.

  • y: Integer

    Returns the y-coordinate of the click.

  • elementId: String

    Returns the ID of an SVG element when clicked.

  • origin: String

    "origin" contains the ID of the widget that triggers the click event.

DisabledClick

Triggered when the disabled widget is clicked.

  • origin: String

    "origin" contains the ID of the widget that triggers the click event.

  • hasPermission: Boolean

    hasPermission defines whether the current user has the required rights to access it.

  • horizontalPos: String

    horizontalPos contains the horizontal position from event DisabledClick in pixels (e.g. 10px starting from the viewport).

  • verticalPos: String

    verticalPos contains the vertical position from event DisabledClick in pixels (e.g. 10px starting from the viewport).

EnableChanged

Triggered when the usability of the widget is modified. This can be modified in different ways:
1) With property enable (binding or via widget action SetEnable)
2) With property permissionOperate
3) With the usability properties (enable, permissionOperate, OPC UA variable) of the surrounding container widget
4) With a bound OPC UA variable whose write permissions are changed.

  • value: Boolean

    "value" defines the usability of the widget. If true is returned, the widget can be operated; if false, it cannot be operated.

VisibleChanged

Triggered when the visibility of the widget is modified. This can be modified in different ways:
1) With property visible (binding or via widget action SetVisible)
2) With property permissionView
3) With the visibility properties (visible, permissionView) of the surrounding container widget

  • value: Boolean

    "value" defines the visibility of the widget. If true is returned, the widget is displayed; if false, it is not displayed.

OnDragEnter

Triggered when a dragged widget is moved over the target widget.

  • widgetId: String

    widgetId contains the ID of the widget that performed the drag operation.

  • contentId: String

    contentId contains the ID of the piece of content from whose widget the drag operation was performed.

OnDragLeave

Triggered when a dragged widget is dragged away from the target widget.

  • widgetId: String

    widgetId contains the ID of the widget that performed the drag operation.

  • contentId: String

    contentId contains the ID of the piece of content from whose widget the drag operation was performed.

OnDrop

Triggered when a dragged widget is released on the target widget. The definition of drag-and-drop and how these events can be used is explained here.

  • widgetId: String

    widgetId contains the ID of the widget that performed the drag operation.

  • contentId: String

    contentId contains the ID of the piece of content from whose widget the drag operation was performed.

FocusIn

Triggered when a widget gets the focus.

 

FocusOut

Triggered when a widget loses the focus.
